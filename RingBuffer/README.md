# C++11 Ring Buffer

一个高性能的无锁线程安全环形缓冲区，专为单生产者单消费者模型设计。

## 特性

-  **无锁设计**：使用原子操作，无需互斥锁
-  **高性能**：避免缓存行伪共享，支持批量操作
-  **线程安全**：适用于单生产者单消费者场景
-  **头文件库**：易于集成，只需包含一个头文件
-  **类型安全**：支持任何可移动构造的类型
-  **零拷贝**：使用移动语义和placement new


## 创建构建目录
mkdir build && cd build

# 配置
cmake ..

# 编译
make

# 运行示例
./examples/basic_example

## API 参考

### 主要方法

- `push(value)`: 入队操作
- `pop(value)`: 出队操作
- `push_batch(data, count)`: 批量入队
- `pop_batch(data, count)`: 批量出队
- `peek(value)`: 查看队首元素
- `empty()`: 检查是否为空
- `full()`: 检查是否已满
- `size()`: 获取当前元素数量
- `available()`: 获取剩余空间
- `capacity()`: 获取缓冲区容量

### 模板参数

- `T`: 元素类型（必须可移动构造）
- `Capacity`: 缓冲区容量（必须是2的幂）

## 性能优化

1. **批量操作**: 使用 `push_batch` 和 `pop_batch` 减少原子操作开销
2. **缓存友好**: 自动对齐到缓存行边界
3. **内存序优化**: 使用正确的最小内存序

## 限制

- 仅适用于**单生产者单消费者**模型
- 容量必须是2的幂
- 类型T必须支持noexcept移动构造
